# 클라이언트 프로그램 설치

설치 프로그램을 더블클릭 해서 설치를 진행한다.







# Login


  • User ID와 Password는 Auth Manage 에서 등록하며, 초기 Password는 User ID와 동일하다.
  • 최초 로그인 시에는 패스워드를 반드시 변경해야 합니다.

비밀번호 생성규칙

최소 8자 ~ 최대 16자이내로 입력합니다. 반드시 영문, 숫자, 특수문자가 각 1자리이상 포함되어야 합니다.

# 접속 정보

서버 접속 정보를 관리하기 위해 버튼을 누른다.

  • Name : 접속 서버명
  • Was Url: WAS 서버 URL
  • Server Info: 서버 IP 와 Port

# Client 버전체크

  • 를 눌러 클라이언트 버전을 확인하고 최신 버전을 다운받을 수 있다.
  • 클라이언트 버전체크 및 업데이트는 Login 후에 System 메뉴에서도 가능하다.

# 방화벽 설정

클라이언트 접속을 위해서 두 개의 포트가 오픈되어야 한다.

  • 기본 포트
    • Was : 9020
    • Server : 9021
  • 기본 포트는 서버 설치 과정에서 변경될수 있다.
