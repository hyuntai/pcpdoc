# Import


Export(.json) file 을 Master Job Editor로 불러온다.

TIP

파일 Import 기능은 데이타를 바로 DB에 저장하지 않고, Workspace(화면) 에만 반영한다.
DB에 저장은 해당 리소스에 대한 Lock을 획득하고 Write를 해야 한다.
