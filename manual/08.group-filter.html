# 관리 도구

# Group Order


설정된 그룹 항목의 순서에 따라 화면의 Hierachy 가 결정된다.



  • 버튼을 눌러 그룹오더 설정 화면을 오픈한다.
  • 버튼으로 그룹오더를 생성하고 Grouping 항목을 을 눌러 선택한다. ( 정렬대상에서 제외)
  • 버튼으로 그룹오더의 순서를 변경한다.

# Filter Manage


보려는 Job Diagram 만 집중할 수 있도록 Filter 를 사용하여 일부 Diagram 를 일시적으로 숨길 수 있다.

  • Filter Manage 화면은 버튼 혹은 Tools -> Filter Manage 메뉴를 선택면 열 수 있다.

# Filter Group 추가

Filter 를 등록 하기 위해, 먼저 Filter 가 속할 Filter Group 을 생성 해야한다.

  • 버튼을 눌러 Filter Group 을 추가 한다.



# Filter 추가

실제 사용될 Filter 를 추가한다. 하나의 Filter 에 여러 Filtering 항목을 정의 할 수 있다.

  • 추가하고자 하는 Filter Group 항목에 마우스 우클릭 -> Add Filter 를 선택하여 Filter 를 추가 한다.



# Filtering 항목 편집

Filtering 할 Job 속성 을 추가/삭제 할 수 있다.

  • 좌측 Filter Groups 에서 편집 할 filter 를 선택 한 뒤, 우측 Include Filter 와 Exclude Filter 에서 Filtering 하고자 하는 항목을 편집한다.
    • Include Filter : 표시할 Filtering 항목 정의
    • Exclude Filter : 숨길 Filtering 항목 정의


  • Include Filter 에 항목 추가
    • Include Filter 에서 버튼을 눌러, 신규 레코드를 추가한다.
    • 신규 레코드에서 마우스 우클릭 -> context menu 를 사용하여 항목 편집한다.
    • Operator 필드에서 조건 (AND, OR) 을 선택 할 수 있다. 조건연산은 첫번째 레코드에서 아래로 순차적으로 한다. 마지막 조건은 무시된다.
  • Include Filter 예제
    • Resource 이름이 SYSTEM 이거나 BATCH-SERVER 이고, Server 가 local 인 Job 만 표시.
Field Operator Value Operator
Resource = SYSTEM OR
Resource = BATCH-SERVER AND
Server = local AND
  • Exclude Filter 에 항목 추가 (Include Filter 와 동일)

# Filter Group 선택

  • Filter Manage 화면 Filter Groups 에서 Filter Group 을 선택하고 select Group 버튼을 누르면, Main 화면 상단의 Filter 드롭박스 에 Filter 목록을 선택 할 수 있다.


Filter Group 단위로 main 화면 상단의 Filter 드롭박스에 Filter 목록이 표시된다.
