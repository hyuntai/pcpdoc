# Conditions


실시간으로 Condition 발행 결과를 조회 하는 화면 이다.

  • Condition 화면이 popup 되면 자동으로 오늘날짜의 모든 condition 이 조회 된다.
  • Condition 이름 및 발행 기간을 조건으로 하여 조회 할 수 있다.

    텍스트 박스에 입력한 단어가 포함된 Condition 이름을 조회한다.

  • 사용자가 다시 버튼을 누르지 않아도 변경 내역이 화면에 실시간으로 반영된다.
  • 를 누르면 Condition 을 직접 입력할 수 있는 창이 열린다. 이 때 서버명은 기본적으로 사용자 계정이 입력된다.
  • 를 누르면 선택된 Condition 이 DB에서 삭제가 된다.
