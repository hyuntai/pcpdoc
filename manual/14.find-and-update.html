# Find And Update

Job 을 조회하여 일괄로 수정 할 수 있다.

# Find and Update 순서

  1. MasterJob 화면이 열린 상태에서, 버튼으로 Find and Update 화면을 연다.

TIP

화면은 크게 3가지 영역으로 나뉜다

  • 상단 Find 영역 : 조회할 Job 조건을 입력 영역
  • 가운데 Update 영역 : 수정할 내용 입력 영역
  • 하단 Result 영역 : 조회 및 수정 결과 표시 영역
  1. Find 영역에서 조회할 Job 정보를 입력한다. 입력할 정보는 다음과 같다.
  • Target (조회 대상)
  • Operator (연산자)
  • Value (검색할 내용)

TIP

  • Target : 마우스 오른쪽 버튼을 사용하여 조회대상을 선택 한다.
  • Value : 일부 조회 대상에 대해서는 마우스 오른쪽 버튼으로 Code Help 기능을 사용할 수 있다.

  1. 버튼을 클릭하여 Find 조건을 추가, 버튼을 클릭하여 선택된 조건을 삭제한다.
    조건을 추가 하면 2번을 다시 수행 한다.

  2. 버튼으로 Find 조건에 맞는 Job 을 검색한다. 검색결과는 Result 영역에 표시된다.

  3. Result 영역에서 Update 할 Job 을 선택한다. 버튼으로 전체를 선택하거나, 버튼으로 전체 선택 해제 할수 있다.

  4. Update 영역에서 Update 할 내용을 입력한다. 입력할 정보는 다음과 같다.

  • Target (Update 대상)
  • Operator (연산자)
  • VA ~ VE (Update 할 내용)

TIP

  • Target : 마우스 오른쪽 버튼을 사용하여 Update 대상을 선택 할 수 있다.
  • VA ~ VE : Target 에 따라 VA ~ VE 에 입력 할 수 있는 내용이 정해진다. VA ~ VE 셀에 hint 혹은 default를 확인 할 수 있다.

  1. 버튼클릭하여 Update 를 수행한다.

  2. Result 영역의 Action 필드를 통하여 Update 성공 여부를 확인한다. Action 은 다음과 같다.

    • Find : 조회성공
    • Update : Update 성공

# Find/Update 컬럼

# Find/Update Target

Job Properties (opens new window) 에 있는 내용을 Target 으로 설정 할 수 있다.

# Find Operator

Operator 설명
Is Target 값과 Value 가 똑같다.
Is not Target 값과 Value 가 같지 않다.
Has Target 값이 콤마로 구분된 Value 중 하나와 똑같다.
Not has Target 값이 콤마로 구분된 Value 중 어떤것에도 포함되지 않는다.
Like Value 에 Wild card 로 * 를 사용할 수 있으며, string의 맨앞과 맨뒤만 * 를 입력을 허용한다.
Not like Like 와 반대

# Update Operator

Operator 설명
Assign VA ~ VE 에 정의된 내용을 Target 값 에 입력 한다.
Clear Target 값을 삭제 한다.
Delete if Target 값이 VA ~ VE 와 같으면 해당 row 는 삭제 한다.
Replace VA 에 입력된 string 이 Target 값의 일부나 전체이면, 그 string은 모두 VB로 치환된다.
Template assign Target 값 에 입력된 string 에, 다른 string을 추가 입력 할 수 있다.
Target 값 string 은 ${SRC} 로 표현한다.
예) Target 값이 'XYZ' 이고, VA 가 'my${SRC}-OK' 면, Target 값은 'myXYZ-OK' 로 치환된다
