# Audit Log

PCP Audit log 조회 기능으로 사용자에 의해 발생한 이벤트를 조회할 수 있다. 조회 가능 한 이벤트 Type 은 다음과 같다.

  • Login / Logout
  • Job Action (Hold, Free, Skip/Set OK, RunNow, Rerun, Kill, StopCyclic)
  • Job Property 변경 (Job 수정, 삭제, 삭제 취소)

다음은 Audit log 화면이다. 메뉴 -> Monitor -> Audit Log 를 통해 Open 할 수 있다.
버튼을 클릭하면 조회할 날짜를 선택 할수있고, 버튼 클릭하면 해당 날짜가 조회 된다.

각 컬럼별 의미는 다음과 같다.

컬럼 설명 비고
Time 이벤트 발생 시각
Type 이벤트 Type (Login/Logout, Job Action, Job Property)
User 사용자 계정
Ip 사용자 IP
Tr 이벤트 메시지ID Login/Logout 은 표시안됨
Contents 이벤트 상세 설명
Result 이벤트 결과 (Success/Fail)
Ref 이벤트 저장소 (db/file)

# Audit Log Filter

Audit Log Filter 화면을 통해 Audit log 를 선별적으로 조회 할 수 있다. Audit log 화면에서 버튼을 클릭하면 아래와 같이 Audit Log Filter 화면을 Open 할 수 있다.

Audit Log 화면의 컬럼을 참조하여 입력 및 적용하면된다.
