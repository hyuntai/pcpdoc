# (Virtual box)centos-8 설치

# VB 설정

  1. hostkey 변경(alt+shift))
  2. mouse usb태블릿 으로 변경

# 게스트확장 설치

Reference (opens new window)

# dnf update kernel*
# dnf install https://dl.fedoraproject.org/pub/epel/epel-release-latest-8.noarch.rpm
# yum install gcc kernel-devel kernel-headers dkms make bzip2 perl
# mkdir /media/VirtualBoxGuestAdditions
# mount -r /dev/cdrom /media/VirtualBoxGuestAdditions
# ./VBoxLinuxAdditions.run
# reboot

# 설정에서 공유할 폴더와 속성을 지정한 후에

# sudo mount -t vboxsf VBShare /home/htkim/VBShare

# PowerControl-Plus 설치

# Agent 배포

  • 우선 ngagt 계정으로 배포함

# systemd 서비스 등록

SELinux 가 활성화 되어 있으면, 서비스 등록시 다른 계정의 홈디렉토리에 있는 파일을 Access할 수 없다.

  • 해결-1) SELinux를 Disable 시킨다 (opens new window)
  • 해결-2) ngagt 설치 경로를 root 소유인 directory(ex-/run)로 옮긴다.

# WAS 설치

  • 계정생성: ngjswas, group -> pcpdev(ngagt, htkim)
  • Java설치 확인, 없으면 아래와 같이 설치 --> tobe: 배포패키지에 openjdk-1.8을 포함시켜야 겠음.
dnf install java-1.8.0-openjdk.x86_64

# 방화벽 설정

firewall-cmd --permanent --add-port 9020-9024/tcp
firewall-cmd --permanent --add-port 5432/tcp
firewall-cmd --reload

# 서버 빌드

dnf install gcc-c++.x86_64  
dnf install libcurl-devel.x86_64
// 그 밖에도 추가적으로 필요한 package 설치할 수 있음
